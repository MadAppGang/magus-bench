# Archive

Completed one-off studies, preserved for reference.

When a benchmark or research study is wound down — either because it's been superseded, its findings have been folded into mainline code, or the experiment is simply done — it moves here rather than being deleted. Each entry is dated and accompanied by a short README explaining what it was and why it's archived.

## The rule

Archive, don't delete. Even dead experiments document a path that was explored; recreating the context months later costs far more than keeping the directory around.
